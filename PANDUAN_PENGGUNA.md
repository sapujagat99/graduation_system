# Panduan Pengguna - Sistem Penilaian Kelulusan Otomatis

Selamat datang di  Sistem Penilaian Kelulusan Otomatis . Dokumen ini akan memandu Anda untuk menggunakan aplikasi ini.

## Cara Menggunakan Aplikasi

1.   Akses Aplikasi 
    *   Buka browser internet Anda (Google Chrome, Mozilla Firefox, Microsoft Edge, dll).
    *   Kunjungi alamat / URL aplikasi  Sistem Penilaian Kelulusan Otomatis .

2.   Isi Data Mahasiswa 
    *   Pada kolom  Nama Mahasiswa , ketik nama lengkap mahasiswa (Contoh: Budi Santoso).
    *   Pada kolom  NIM , ketik Nomor Induk Mahasiswa (Contoh: 12345678).

3.   Isi Nilai 
    *   Masukkan nilai angka (0 - 100) pada kolom yang tersedia:
        *    Nilai Tugas (30%) : Masukkan nilai rata-rata tugas.
        *    Nilai UTS (30%) : Masukkan nilai Ujian Tengah Semester.
        *    Nilai UAS (40%) : Masukkan nilai Ujian Akhir Semester.

4.   Hitung Hasil 
    *   Setelah semua data terisi, klik tombol  "Hitung Hasil" .
    *   Sistem akan secara otomatis menghitung Nilai Akhir sesuai bobot yang ditentukan.

5.   Lihat Hasil 
    *   Kartu hasil akan muncul di bagian bawah.
    *   Anda dapat melihat:
        *    Nilai Akhir : Angka desimal hasil perhitungan.
        *    Grade : Indeks huruf (A, B, C, D, atau E).
        *    Status : Keterangan LULUS atau TIDAK LULUS.

6.   Reset Data 
    *   Untuk menghitung mahasiswa lain, klik ikon "Reset" (tanda panah melingkar) di pojok kanan atas kartu hasil untuk mengosongkan formulir.

## Kriteria Penilaian

Sistem menggunakan aturan penilaian sebagai berikut:

 Perhitungan Nilai Akhir: 
*   (Nilai Tugas x 30%) + (Nilai UTS x 30%) + (Nilai UAS x 40%)

 Kategori Grade: 
| Grade | Rentang Nilai |
| :--- | :--- |
|  A  | 85 - 100 |
|  B  | 70 - 84.99 |
|  C  | 55 - 69.99 |
|  D  | 40 - 54.99 |
|  E  | 0 - 39.99 |

 Kriteria Kelulusan: 
*    LULUS : Grade A, B, atau C.
*    TIDAK LULUS : Grade D atau E.
